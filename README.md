## About the app

This is a food delivery's courier app  created for managing couriers

It allows you to order get real time orders from database and update your current location

You first need to register to use the app.  For registration and log in there is used FireBase.

This application was originally created by Shalva Lomtadze and Giga Maisuradze

## Features

The android app lets you:
- check for active orders.
- get delivery destination location.
- have navigation to destination so you don't get lost.
- chat with client that you have arrived.
- service which updates your location based on your current one

Application requires the following permissions:
